Proiect Git¶

Cum să fac repede o copie de rezervă a proiectului Git?¶

Folosind comanda git-clone cu opțiunea (pentru orice eventualitate) --no-hardlinks:

git clone --no-hardlinks <URL proiect Git sursă> <dosar destinație>
