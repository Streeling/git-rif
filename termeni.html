obiect git [en. git object]
Un obiect git poate fi: un blob, un arbore, o comitere sau o etichetă; orice altă entitate în afara acestei liste nu este obiect git.
blob
O clecție de octeți care cel mai des reprezintă conținutul unui fișier.
arbore
O structură care conține refințe către alți arbori și blob-uri; poate fi privit ca un dosar.
comitere
Conține o singură referință către un arbore.
etichetă
Un nume asociat unei comiteri.
